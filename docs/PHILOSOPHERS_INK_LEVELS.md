- 버전: 1.0 (2026-07-15)
- 범위: 런칭 77레벨 전체의 구조·난이도·레벨별 명세·검증 기준

## 1. 왜 77인가 — 구조의 근거

77 = **7 × 11**. 연금술의 7대 작업(Seven Operations)을 11레벨 간격의 마일스톤으로 배치한다.

- **레벨 11·22·33·44·55·66·77 = 작업(OPERATIO) 레벨** — 직전 10레벨에서 배운 것을 종합하는 시험.
- 챕터 경계는 11의 배수에 정렬: **1–11 니그레도 / 12–33 알베도 / 34–55 키트리니타스 / 56–77 루베도.**
- 튜토리얼 챕터(니그레도)가 가장 짧은 것은 의도다 — 첫 마법(서리 룬, 레벨 17)까지 빨리 데려간다 (조기 승리 원칙).

| 챕터 | 레벨 | 수 | 작업 레벨 | 신규 요소 |
|---|---|---|---|---|
| 1 니그레도 | 1–11 | 11 | 11 하소 | 석필, PRIMA, 중력 반전 |
| 2 알베도 | 12–33 | 22 | 22 용해 · 33 분리 | WATER/ICE, 온도 존, 서리 룬, 상태 플라스크, 포탈, 변성 게이트 |
| 3 키트리니타스 | 34–55 | 22 | 44 결합 · 55 발효 | 화염 룬, STEAM, 재/순수, 유한 방출 |
| 4 루베도 | 56–77 | 22 | 66 증류 · 77 응고 | LAVA, STONE, 용암+물 반응 |

작업 레벨 규격: 레벨 셀에 골드 링 표시(셸 골드 희소성 원칙의 예외 승격), 전용 스팅어 사운드,
클리어 시 작업 인장(문양) 획득 연출. 작업 레벨은 힌트 광고 비활성(시험의 존엄).

## 2. 난이도 모델

### 2.1 난이도 노브 (레벨을 어렵게 만드는 7개 축)

| 노브 | 설명 | 저 → 고 |
|---|---|---|
| K1 목표 부하 | 플라스크 수 × 요구량 | 1개 무조건 → 3개+ 조건부 |
| K2 조건 복잡도 | 무조건 → 물질 지정 → 상태 지정 → 순수(❗) | |
| K3 상전이 깊이 | 클리어에 필요한 최소 전이 횟수 | 0회 → 3회+ (고→액→기→액) |
| K4 잉크 여유율 | 지급 잉크 ÷ 최적해 잉크 | 250% → 135% |
| K5 기믹 결합 수 | 동시에 작동하는 기믹 종류 | 0 → 3+ |
| K6 경로 복잡도 | 필요한 방향 전환·분기 수 | 직선 → 다분기 미로 |
| K7 시간·자원 압박 | 유한 방출, 타이밍 요구 | 없음 → 빡빡한 총량 |

레벨의 난이도 D(1~10)는 노브 조합의 체감 결과값이며, 설계 시 **한 레벨에서 두 개 이상의 노브를
동시에 올리지 않는다** (새 요소 등장 레벨에서는 모든 노브를 내린다).

### 2.2 D밴드의 체감 정의 (플레이테스트 목표치)

| D | 체감 | 목표 시도 수 | 목표 소요 |
|---|---|---|---|
| 1–2 | 손이 먼저 푼다 | 1.0–1.2회 | <40초 |
| 3–4 | 잠깐 멈춰 생각 | 1.2–1.8회 | 40–90초 |
| 5–6 | 한 번은 틀린다 | 1.8–3회 | 1.5–3분 |
| 7–8 | 계획이 필요하다 | 3–6회 | 3–6분 |
| 9–10 | 시험. 종이가 필요할 수도 | 6–12회 | 6–12분 |

## 3. 페이싱 원칙

1. **교육 → 곱셈 → 시험**: 새 요소는 그것만으로 클리어되는 dip 레벨(D −2~−3)로 등장하고,
   직후 1~2레벨에서 기존 요소와 곱해지며, 11배수 작업 레벨에서 시험된다.
2. **사인파 난이도**: 작업 레벨 스파이크 직후는 반드시 휴식 비트 — 챕터 첫 레벨(12·34·56)과
   중간시험 다음 레벨(23·45·67)은 의도적 쿨다운이다.
3. **스파이크 단조 증가**: 작업 레벨의 D는 5→6→7→7→8→9→10으로만 오른다. 뒤 시험이 앞 시험보다 쉬우면 안 된다.
4. **3성 = 한 수 깊은 해법**: 모든 레벨은 "그냥 클리어" 해법과 "잉크를 아끼는 우아한 해법"이 분리되어야 한다.
   1성 클리어가 막히는 레벨은 실패지만, 3성이 쉬운 레벨도 실패다.
5. **표현의 자유**: 정답 선이 유일한 레벨은 금지. 최소 2가지 유효 해법 경로를 에디터에서 검증한다.

## 4. 잉크 예산 · 별점 공식

- 지급 잉크 = 최적해 잉크 × 여유율. 여유율은 D밴드로 결정:

| D | 여유율 |
|---|---|
| 1–2 | 250% |
| 3–4 | 200% |
| 5–6 | 170% |
| 7–8 | 150% |
| 9–10 | 135% |

- ★★★ = 사용량 ≤ 최적해 × 1.15 / ★★ = ≤ 최적해 × 1.6 / ★ = 클리어.
- 최적해 잉크는 에디터의 저작자 플레이 기록 중 최소값으로 저장하고, 내부 테스트에서 갱신되면 자동 반영.

## 8. 검증 · 플레이테스트 기준
