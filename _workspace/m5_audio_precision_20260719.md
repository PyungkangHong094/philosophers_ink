# M5 오디오 정밀 재설계 리포트 (2026-07-19)

담당: shell-ui-engineer (shell-m4). 소유: `lib/audio`.
SSOT: GDD 9.1~9.2(파티클 ASMR·레이어 구조·믹스 계층) + `game-audio`·`ink-shell-design` 스킬.

에셋 없이 flutter_soloud 절차 합성으로 GDD 9 스펙을 1차 완전 구현. **청감은 사용자 몫**(이 환경은
오디오 출력 불가) — 구조·결정성·회귀는 테스트로, 통합은 iOS 시뮬 빌드로 검증했다.

## 1. 아키텍처

- **톤 SFX**: 파형 오실레이터 **풀**(triangle×4·fSquare×3·fSaw×2)을 라운드로빈으로 써서
  겹치는 음이 같은 소스의 주파수를 덮어쓰는 글리치를 막는다(폴리포니). 재생마다 주파수(±22c)·
  볼륨(±12%)·길이(±12%) 지터로 사실상 무한 변주.
- **노이즈 SFX**: `wav.dart`가 화이트노이즈에 1극 로우/하이패스 + 어택/지수감쇠 엔벨로프 + 워블 +
  톤 성분을 얹어 짧은 PCM WAV(수십~백 ms)를 코드 생성 → soloud `loadMem`. 변주 3~5개 사전 생성,
  재생 시 무작위 선택. 파형 enum에 노이즈가 없어 이 방식이 유일.
- **드론 방지(P1 재발 방지) 불변식**: **연속 루프는 BGM(기본 OFF)뿐.** SFX·그레인·앰비언트는
  전부 ≤300ms 원샷이라 "3초 이상 동일 톤"이 구조적으로 불가능. (테스트로 고정.)
- 생명주기: dispose·앱 백그라운드에서 `stopAll()`(BGM 정지). 그레인은 원샷이라 자동 소멸.

### 파일
신규 `lib/audio/wav.dart`(순수 노이즈 합성·WAV 인코딩). 재작성 `sound_tokens.dart`(전 파라미터),
`audio_service.dart`(계약 확장), `soloud_audio_service.dart`(구현). 수정 `play_screen`(onPhaseChange·
setBgmChapter·3밀도 앰비언스·flask progress), `settings_controller`/`settings_screen`(BGM 토글),
`app`(configure bgmEnabled).

## 2. 사운드별 설계표 (레이어 · 변주 · 믹스)

| 사운드 | 소스/합성 | 레이어(어택/바디/테일) | 변주 | 믹스 | 트리거·특기 |
|---|---|---|---|---|---|
| UI 탭 | fSquare + triangle 배음 | 클릭 + 2배음 반짝 | ∞(지터) | 0dB | 버튼·셀·병선택 |
| 획(stroke) | triangle | 짧은 틱 | ∞(지터) | 0dB | 배치 시, 45ms 스로틀 |
| 착수 틱 | triangle | 틱 + **카운트업 피치 램프** | ∞ | 0dB | 상별 피치(고체 G5/액체 D5/기체 B4) + 채움 0→1에 +12반음 상승. 70ms 스로틀 |
| 클리어 스팅어 | triangle 아르페지오 | C5·E5·G5 상승 | ∞ | 0dB | + BGM 덕킹 |
| 작업 스팅어 | fSquare 아르페지오 | G4·C5·E5·G5 하강풍성 | ∞ | 0dB | OPERATIO 전용 + 덕킹 |
| 실패 | fSaw 하강 | E4·C4 | ∞ | 0dB | 오염 시 |
| 결빙 crackle | 노이즈+톤핑, 밴드패스 | 바삭 어택+빠른 감쇠 | 3 | −3dB | ICE 생성(onPhaseChange), 위치·밀도 스로틀 |
| 증발 puff | 로우패스 노이즈 | 느린 어택+공기 테일 | 3 | −3dB | STEAM 생성, 스로틀 |
| 반응 sizzle | 노이즈+워블 | 지글 지속 | 3 | −3dB | LAVA→STONE, 스로틀 |
| 파티클 그레인 | 초단 로우패스 클릭 | 클릭 | 5 | −9dB | 동적 밀도 확률 발사, 밀도→피치(0.8~1.3배) |
| 물 앰비언트 | 로우 노이즈 워블 | 워블 | 4 | −9dB | WATER 셀 수 비례 |
| 증기 앰비언트 | 하이패스 히스 | 히스 | 4 | −9dB | STEAM 셀 수 비례 |
| BGM 패드 | sine 화음 루프 | 근음+5도+옥타브 | 챕터4색 | −12dB | **기본 OFF**·설정 토글, 클리어 덕킹 |

- **믹스 계층(GDD 9.2)**: 이벤트 0dB > 상전이 −3dB > 그레인/앰비언트 −9dB > BGM −12dB. 전부
  `SfxMix`(dB→선형) 토큰. 매직 넘버 0.
- **상전이 스로틀**: 틱당 다수 이벤트 → 100ms 윈도 내 최대 2회 재생, 초과분은 생략(윈도 내
  후속은 볼륨 +15%/회로 밀도감만 가산).
- **파티클/앰비언트**: PlayScreen이 그리드를 ~5Hz로 1회 스캔해 (동적/물/증기) 3밀도를 뽑아
  `setAmbience`로 전달 → 밀도 비례 개수의 그레인을 확률 발사(연속 루프 아님).

## 3. sim 연동 (결정성 무영향)

- `GameState.onPhaseChange(materialFrom, materialTo, x, y)` 훅(sim-engineer 구현, 관찰 전용)을
  PlayScreen이 구독 → materialTo로 분기: ICE→crackle, STEAM→puff, STONE→sizzle.
- 착수는 기존 `LevelSession(onSettle)` — flaskIndex로 진행도 계산해 피치 램프.
- lib/sim 무수정. 콜백은 RNG·그리드를 건드리지 않아 재시작 결정성 유지(sim 계약).

## 5. 사용자 청감 평가 체크리스트

시뮬레이터/실기에서 아래를 귀로 확인해 주세요(부착 run 세션 hot reload 가능).

**기본(사운드 ON, BGM OFF)**
- [ ] 레벨 시작~드로잉 중 배경 "지잉" 드론 없음(정적).
- [ ] 선을 그을 때 부드러운 틱, 빠르게 그어도 거슬리는 버즈 없음.
- [ ] 플라스크가 차오를 때 착수 틱의 **피치가 점점 올라가** 만족감(카운트업).
- [ ] 얼음이 얼 때 바삭한 결빙음, 증기 생길 때 공기 빠지는 소리(레벨 15~21 등).
- [ ] 입자가 쏟아지고 쌓일 때 성긴 그레인 텍스처(과하지 않게).
- [ ] 클리어 시 상승 스팅어, 작업 레벨(11·22…)은 더 낮고 풍성한 전용 스팅어.
- [ ] UI 버튼·병 선택음이 경쾌하고, 같은 소리도 미묘하게 매번 다름(반복감 없음).
- [ ] 화면을 나가거나 앱을 백그라운드로 보내면 즉시 무음.

**BGM ON(설정 → 배경 음악)**
- [ ] 챕터별로 근음이 다른 낮은 패드가 아주 작게 깔림(−12dB).
- [ ] 클리어 스팅어 동안 BGM이 잠깐 작아졌다(덕킹) 돌아옴.
- [ ] 만약 패드가 거슬리면 OFF로 두면 됨(기본 OFF).

**볼륨**
- [ ] 설정 볼륨 슬라이더가 전체 크기를 조절.

## 6. 한계·이연

- **BGM 정적 패드**: 절차 sine 화음 루프라 움직임이 적다(청감 미검증). 기본 OFF + 토글 +
  −12dB + 덕킹까지만 1차 구현. LFO 볼륨 흔들기·화성 전개는 폴리시 이연. 품질이 나쁘면 그대로
  OFF 두면 되고, 좋으면 기본 ON 검토.
- **노이즈 질감 튜닝**: 필터 계수·엔벨로프는 청감 없이 설정한 초기값. 사용자 피드백으로
  `sound_tokens.dart`의 `GrainKit`/`GrainPlay` 한 곳에서 조정.
- **BGM 화성·챕터 팔레트 매핑**은 근음만 다르게 한 근사. 챕터 감정선 반영은 폴리시.
- 커밋하지 않음.
