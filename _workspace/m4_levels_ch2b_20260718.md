### 023 문 (D3, portal 교육 dip)
- 의도: WATER를 중앙 상단에서 방출. 좌측 포탈 입구로 유도(석필 경사) → 우측 상단 출구로 순간이동 → 플라스크 낙하.
- 경로 A: 완만한 대각 경사 1획으로 물을 좌측 입구에 몰아넣기. 경로 B: 짧은 계단식 2획으로 유도.
- 잉크: optimal chalk 18 / 예산 chalk 48. (dip 관대)

### 024 두 개의 문 (D5)
- 의도: WATER(좌)·PRIMA(우) 2방출 + 포탈 2쌍이 교차 수송. 물→우 플라스크(WATER 조건), 프리마→좌 플라스크(PRIMA 조건). 중앙 분리벽으로 하단 혼입 차단.
- 경로 A: 각 방출 아래 짧은 넛지 경사로 입구에 정렬. 경로 B: 방출 밴드 바로 아래 받침선으로 낙하각 보정.
- 잉크: optimal chalk 44 / 예산 chalk 75 (170%).

### 025 변성의 문 (D4, variance_gate 교육 dip)

- 경로 A: 선반 위 안쪽 경사로 물 모으기. 경로 B: 게이트 아래 직접 깔때기 1획.
- 잉크: optimal chalk 16 / 예산 chalk 48. (dip 관대)

### 026 되돌림 (D5)
- 의도: PRIMA → 게이트1(→WATER) → y140 벽의 6폭 틈(액체만 통과, 입자는 퇴적) → 게이트2(→PRIMA) → PRIMA 조건 플라스크. 물성 왕복.
- 경로 A: 틈 위 V자 깔때기로 물 집중. 경로 B: 넓은 받침 후 중앙 유도.

### 027 물방울 서약 (D5)
- 의도: 좌측 PRIMA 방출. 우측 절반(x60~160)에만 변성 게이트. 액체 조건 플라스크는 우측. 프리마를 게이트 구간까지 밀어 물로 바꿔야만 액체 조건 충족(게이트 경유 강제).
- 경로 A: 좌→우 장거리 램프 1획. 경로 B: 2단 계단으로 게이트 진입.
- 잉크: optimal chalk 50 / 예산 chalk 85 (170%). (장거리 램프라 예산 상단)

### 028 갈림길 (D6)
- 의도: 중앙 스플리터 능선(지형)으로 WATER 흐름 2분할. 좌측 강하에 서리선 → ICE 결빙 → 고체 플라스크. 우측은 물 그대로 → 액체 플라스크. 한 방출, 두 조건.
- 경로 A: 좌측만 서리선, 양쪽 석필 경사. 경로 B: 스플리터에 기대 최소 석필 + 서리 집중.
- 잉크: optimal chalk 28·frost 26 / 예산 chalk 48·frost 44 (170%).

### 029 쏟아지는 천장 (D6, gravity_flip)
- 의도: 하단 분지에 물을 고인 뒤 중력 반전 → 고인 물이 천장 플라스크로 역류. 반전 타이밍이 관건.
- 경로 A: 분지 보강(석필) 후 충분히 고이면 1회 반전. 경로 B: 얕게 여러 번 반전 왕복(석필 절약).
- 잉크: optimal chalk 40 / 예산 chalk 68 (170%).

### 030 원격 결빙 (D6)
- 의도: WATER를 포탈로 협곡 상공(원격)으로 보냄. 출구 아래 서리선을 미리 깔아 물을 ICE 다리로 굳힘. 좌측 PRIMA를 석필 램프로 얼음 다리 위에 올려 우측 플라스크(PRIMA 조건)까지 굴린다.
- 경로 A: 협곡 폭 전체에 서리선 + 프리마 램프. 경로 B: 얼음 슬립(안식각)을 이용해 짧은 서리선으로 다리 완성.
- 잉크: optimal chalk 36·frost 26 / 예산 chalk 61·frost 44 (170%).

### 031 잉크 기근 (D7, chalk 0)
- 의도: 석필 0. 서리선만으로 낙하하는 물을 ICE로 굳혀, 얼음 더미(슬립으로 퍼짐)만으로 협곡을 건너는 램프/다리를 세워 고체 플라스크에 얼음을 쌓는다.
- 경로 A: 협곡 위 대각 서리선 1~2개로 얼음 램프. 경로 B: 얇은 수직 서리선 여러 개로 계단식 얼음 더미.
- 잉크: optimal frost 34 / 예산 frost 51, chalk 0 (150%). K1/K4 극단 제약.

### 032 이중 관문 (D7, 3중 결합)
- 의도: PRIMA 방출을 중앙 분리벽으로 분기. 좌: 그대로 고체 플라스크. 우: 포탈(입구 좌측→출구 우측 상단)로 벽 넘어 → 출구 아래 게이트(→WATER) → 액체 플라스크. 포탈+게이트+상태 플라스크 결합.
- 경로 A: 상단에서 좌우 분배 후 우측만 포탈 진입. 경로 B: 전량 포탈→게이트로 액체 우선 채우고 잔량으로 고체.
- 잉크: optimal chalk 50 / 예산 chalk 75 (150%).

### 033 ◈ 분리 (D7, 작업 레벨 Separatio)
- 의도: PRIMA+WATER 동일 밴드 혼합 방출. 물(액체·확산)과 프리마(입자·퇴적)의 거동 차이를 이용해 좌 WATER 조건·우 PRIMA 조건 플라스크로 완전 분리. 스플리터 능선 + 서리(물 고정)로 정렬. 중력 반전은 대체 해법용 도구(재분배).
- 경로 A(물리 분류): 능선으로 물은 확산시켜 좌측 원거리, 프리마는 근거리 퇴적 우측. 서리로 물 절반을 얼려 위치 고정 후 분배.
- 경로 B(반전 활용): 물을 한쪽에 고이게 한 뒤 중력 반전으로 프리마만 위로 털어내 분리 → 반전 복귀로 각 플라스크에 안착.
- 잉크: optimal chalk 54·frost 28 / 예산 chalk 80·frost 42 (150%, 작업 레벨 타이트 상단). `hint_stroke: null`.

## 설계표와 달리한 점

- **026 되돌림**: 원안의 "상태 지정(고체) + 왕복"에서 상태 플라스크 조건을 빼고 `material: PRIMA`만으로 판정.
  사유: PRIMA는 입자=고체상이라 material 지정만으로 고체 결과가 강제되고, 한 레벨 노브 2개 동시 상승(K2 상태 + K5 게이트 왕복) 회피(스킬 규칙 4). 핵심 아이디어(역방향 게이트 왕복)는 유지.
- 그 외 10개 레벨은 설계표 이름·신규 요소·핵심 아이디어·D값을 그대로 구현.
